Welcome to the Artistic Object Detection Project

This project allows you to detect objects in real-time using a webcam and machine learning models. The goal is to help visualize AI-driven object detection in a user-friendly way. You can interact with the system and observe how it identifies various objects!

Created by Nahom, a student who is passionate about AI and machine learning.
